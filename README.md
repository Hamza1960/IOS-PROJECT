# MindTrack: Daily Wellness Journal iOS App
## Student: Hamza Patel
### Course: CSC-371/471 - DePaul University

## Overview
MindTrack is a daily wellness journal app for iOS built with UIKit. It helps users track their daily mood, energy levels, and thoughts in a visual and interactive way.

## Features
- Daily mood and energy tracking with tap selections
- Quick note writing for daily thoughts and reflections
- Visual mood charts using **Quartz 2D** graphics
- Interactive timeline with **multi-touch** swipe and pinch gestures
- Customizable settings (reminders, themes)
- Core Data persistence for all journal entries

## Screens
1. **Home Dashboard** - Today's summary, weekly mood average, quick log button
2. **Daily Entry** - Mood selector (5 emoji buttons), energy slider, notes text area
3. **Mood Charts** - Quartz 2D line/bar charts with pinch-to-zoom and swipe gestures
4. **Journal History** - Scrollable table of past entries with swipe-to-delete
5. **Settings** - Reminders, theme toggle, data management

## How to Run
1. Open `MindTrack.xcodeproj` in Xcode
2. Select a simulator (e.g., iPhone 15)
3. Press **Run** (`Cmd + R`)
